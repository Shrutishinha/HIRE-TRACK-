# 🚀 HireTrack

HireTrack is a full-stack job tracking platform that helps manage job applications, candidates, and hiring workflows efficiently.

## ✨ Features
- Dashboard for tracking applications
- Resume parsing & analysis
- Kanban pipeline for job stages
- Job feed integration
- AI-based insights

## 🛠️ Tech Stack
- React + Vite + Tailwind CSS
- Node.js + Express
- MongoDB

## ⚙️ Run Locally
```bash
npm install
cd backend && npm install
npm run dev:all
